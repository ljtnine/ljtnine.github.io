    • openwrt利用privoxy配合shadowsocks开启http代理

    为方便ie或移动设备使用代理,openwrt上需开启http代理服务.

    1.安装privoxy

    opkg install privoxy

    2.配置privoxy

    vi /etc/privoxy/config

    3.启动privoxy

    /etc/init.d/privoxy start

    4.将ie代理设置为192.168.10.1:8118 即可使用http代理
